# Как перенести эти заметки в Obsidian

## Важно

- **Obsidian** — это хранилище **текстовых заметок** (Markdown).  
- **Код** дашборда (`web/`, `node_modules`, сборки) в хранилище **не кладём** — Obsidian будет тормозить, путаница с версиями и git.

**Нормальная схема:** в Obsidian — описания, планы, ссылки; код — в `Neral-Agrocenter-ai` (или git).

## Вариант A — скопировать заметки в уже существующее хранилище

1. Скопируйте **только файлы `.md`** из папки  
   `Neral-Agrocenter-ai/obsidian/`  
   в вашу папку хранилища Obsidian (куда у вас лежат заметки).
2. В заметке [[Neral — репозиторий код]] поправьте путь к проекту на диске, если у вас другой.

## Вариант B — открыть эту папку как отдельное хранилище

1. В Obsidian: **Open folder as vault** → выберите  
   `...\Neral-Agrocenter-ai\obsidian`  
   (только подпапку `obsidian`, не весь репозиторий).
2. Тогда заметки живут рядом с репо; ссылки на код можно делать относительными: `../web/README.md` (в Obsidian внешние файлы открываются по клику не всегда — удобнее дублировать путь в заметке).

## Вариант C — симлинк только заметок (для продвинутых)

Оставить vault где привыкли, а эту папку `obsidian` подключить симлинком внутрь vault — одна копия файлов на диске.

---

После импорта откройте **[[Neral — стартовая точка]]** как главную карту по проекту.
