# Веб-дашборд группы (прототип)

Стек: **Vite 6**, **React 19**, **TypeScript**, **Tailwind CSS v4**, **React Router**, **Recharts**.

## Запуск

```bash
cd web
npm install
npm run dev
```

Откройте в браузере адрес из консоли (обычно `http://localhost:5173`).

## Данные (моки)

| Файл | Назначение |
|------|------------|
| `src/data/companies.ts` | Юрлица, выручка/прибыль/активы (как в ресёрче) |
| `src/data/financeMocks.ts` | Касса, дебиторка, кредиторка по годам (2024/2025) |

Подмена на **API** — следующий этап (`VITE_API_URL`, `fetch`).

## Скелет интерфейса

- **Шапка страницы** (`PageShell`): год, квартал (подпись; нарезка цифр по кварталам — позже), вход демо (`/login`), выход.
- **Разделы меню:** сводка, финансы (касса / дебиторка / кредиторка), алерты (заглушка под план), юрлица, админ (настройки/доступы).
- **Карточка компании:** вкладки «Обзор» и «Финансы (мок)» (`?tab=finance`).
- **CSV:** кнопки на страницах дебиторки и кредиторки (клиентская выгрузка для примера).

## Контексты

- `src/context/AuthContext.tsx` — демо-пользователь в `localStorage`.
- `src/context/DashboardFiltersContext.tsx` — год и квартал для экранов.
