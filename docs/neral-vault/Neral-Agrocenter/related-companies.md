# Связанные компании ООО «Нерал-Агроцентр»

## 7. Связанные по адресу/учредителю — без детализации

По rusprofile у Агроцентра: **5 активных + 13 ликвидированных** связанных юрлиц (по адресу или учредителю). Детальный пофирменный extract требует Контур.Фокус или СПАРК. Топ-кандидаты для углубления:
- Прочие СХП группы (Чишмы и Буздяк — выше).
- Дистрибьюция «Милк Трейд».
- Производственные «Башкирское мороженое», «Серебряный снег» (ИНН не подтверждены в этой сессии).

## 8. Что значит scope группы

**Группа НЕРАЛ — это не одна ферма. Это вертикально интегрированный агро-FMCG-холдинг:**

| Звено | Юрлицо | Масштаб |
|---|---|---|
| Земля и растениеводство | СХП «Нерал-Чишмы» + «Нерал-Буздяк» | ~1,3 млрд ₽ выручки совокупно |
| Услуги полям | Нерал-Агроцентр | 35 млн ₽ |
| Молоко-сырьё | Те же СХП (КРС молочный) | – |
| Переработка молока | Белебеевский молкомбинат (49%) | ~5+ млрд ₽ |
| Заморозка/дистрибуция | АО «Башкирский холод» | – |
| Конечные FMCG-бренды | «Башкирское мороженое», «Серебряный снег» | – |
| Прочее | Мастер окон, шампанские вина | – |
| Управление | Концерн «Интел Ко» | 180 млн ₽ выручки + 449 млн ₽ прибыли (=дивиденды) |

**Импликация для AI-стратегии:** см. [[automation-roadmap]] — дашборд директора должен видеть все звенья, не только Агроцентр. Каждое звено генерирует свои метрики (поле → молоко → сыр → дистрибуция → касса).
