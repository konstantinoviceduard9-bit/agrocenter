### C3. Уровень consolidation (унифицированных систем)

**На текущей стадии (по public-данным + анализу):**

| Функция | Уровень consolidation | Реальность |
|---|---|---|
| ERP (1С) | Скорее всего, **per-entity** | Каждая компания — своя инсталляция, без единого master-каталога |
| HR | Per-entity (разные директора, разные ТД) | Возможно общий HR-аутсорсинг через Интел Ко |
| Юридический офис | Скорее **shared** (через Интел Ко) | ОКВЭД Интел Ко включает юр.услуги |
| Бухгалтерия | Per-entity, но возможно общий аутсорсинг | Каждое юрлицо сдаёт самостоятельно |
| Treasury / банк-операции | Per-entity | Раздельные счета, нет единого cash pool по public-данным |
| ИТ-инфра | Per-entity, без shared data lake | Подтверждается фактом, что в Агроцентре нет data warehouse |
| Безопасность / комплаенс | Per-entity | – |
| Логистика | Частично shared через Милк Трейд (FMCG) | Агро-логистика отдельно от FMCG |

**Вывод:** Группа **слабо консолидирована на уровне ИТ-систем**. Это:
- **Минус:** дублирование функций, нет единого «правда о метриках»
- **Плюс:** огромный TAM для consolidation savings — group-wide ERP/dashboard/treasury может дать 10-15% EBITDA-прироста на shared services alone.

### C5. Стратегические impl для AI-стратегии

#### Implication 1 — **Group-wide ERP — это $XX млн TAM, не $1 млн**

Если бы это была одна 35-млн ₽ компания, ROI на ERP был бы низким. Но **14 млрд ₽ группы** меняет расчёт: даже 5% операционной экономии на shared services = **700 млн ₽/год**.

#### Implication 2 — Агроцентр = **«троянский конь» для group-wide pilot**

Малая компания (35 млн ₽, 13 чел.) с мотивированным CEO (Бердина, 14 лет в должности) — идеальный sandbox. Шампион → доказательство → масштабирование через Узбекова (гендиректор Интел Ко) на 12+ других юрлиц.

#### Implication 3 — Белебеевский молком — **самый горячий target по ROI**

10 млрд ₽ выручки, новый кризисный CEO (Шаманов, 5 мес), банкротство на 1,9 млрд. AI-restructuring dashboard для него — потенциально **highest-value initiative в группе** (выше Агроцентрового дашборда по абсолюту).

#### Implication 4 — Семейная группа = единый stakeholder map

Исаев Э.Ф. (51%) + семья Шигаповых (40% через 2 человека + 75% Нерал-Строй) + Козлов (9%) — это 4-5 ключевых физлиц, контролирующих все 25 юрлиц. **Не loose alliance, а tight family-controlled empire**. Это:
- упрощает decision-making (один центр)
- но создаёт key-person risk (Исаев — 70 лет+, его наследование критично)

#### Implication 5 — Тренд снижения выручки 2021→2025 (-18%) = триггер для трансформации

Группа **теряет 3 млрд ₽ выручки за 4 года**. Это не «развивающийся» бизнес — это **зрелый, под давлением, ищет efficiency-плечи**. AI/automation идеально ложится: не «вырасти на 30%», а «удержать маржу при сокращающейся выручке через -20% операционных затрат».
