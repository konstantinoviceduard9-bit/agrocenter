# Установка «Нерал-Матрикс» на телефон (PWA)

Без Google Play и App Store: одна иконка на рабочем столе, полноэкранное приложение.

## Ссылка для сотрудников

```
https://konstantinoviceduard9-bit.github.io/agrocenter/matrix/login
```

После установки иконка открывает **вход** (не общий пульт в браузере).

## Android (Chrome)

1. Откройте ссылку в Chrome.
2. Меню **⋮** → **«Установить приложение»** или **«Добавить на главный экран»**.
3. Подтвердите. На рабочем столе появится **«Матрикс»**.

## iPhone (Safari)

1. Откройте ссылку именно в **Safari** (не во встроенном браузере Telegram/WhatsApp).
2. **Поделиться** (квадрат со стрелкой) → **«На экран Домой»** → **«Добавить»**.

## Вход

- PIN демо: **`1`**
- Нажмите **своё имя** в списке (PIN один у всех).

## Задачи от руководства

- Назначение **на том же телефоне** — сразу в «Мои задачи».
- С **компьютера** — после «Назначить» нажмите **«Отправить ссылку»** и перешлите водителю/доярке в мессенджер.
- Автоматическая синхронизация без ссылки — когда подключим сервер.

## Выполнение задачи

- Сотрудник нажимает **«Выполнена»** → подтверждение → уведомление руководству.
- Руководитель видит список в **«Сотрудники»** → блок **«Уведомления руководству»**.
- С другого телефона: после выполнения — **«Сообщить руководству»** (ссылка в мессенджер).

## Для IT

- Сборка: `web-matrix` + `vite-plugin-pwa`, `start_url: ./login`, `display: standalone`.
- Деплой: тот же GitHub Pages, каталог `/matrix/`.
- Офлайн: кэш статики через service worker; данные пока демо (localStorage).
